# Variables de Entorno

Actualmente el proyecto **no utiliza variables de entorno**. Toda la configuración
se define en archivos YAML dentro del directorio `commands/`.

## Futuras variables de entorno (planeadas)

| Variable | Propósito | Fase |
|---|---|---|
| `AGENTE_LANG` | Idioma del agente (es/en) | Fase 2 |
| `AGENTE_LLM_URL` | URL del LLM local (Ollama) | Fase 2 |
| `AGENTE_LLM_MODEL` | Modelo del LLM (ej. llama3) | Fase 2 |
| `AGENTE_LOGS_DIR` | Directorio de logs | Fase 2 |
| `AGENTE_CONFIG_DIR` | Directorio de configuración YAML | Fase 2 |
